First install additional requirements (requirements.txt).
In order to run Caffe examples, you need to prepare data.
Go data/ dir and run make_caffe_data.sh script and then it is done.
If you are using provided AMI, you can skip these steps.

To run caffe examples, run rungpu.sh for GPU, runcpu.sh for CPU
and runboth.sh for both CPU and GPU.
